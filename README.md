# 🧙‍♂️ พ่อมดโค้ดดิ้ง | The Coding Oracle

[![Thai Language](https://img.shields.io/badge/Language-Thai-blue.svg)](#-ภาษาไทย)
[![English Language](https://img.shields.io/badge/Language-English-red.svg)](#-english)

เว็บบล็อกและแอปพลิเคชันทำนายดวงชะตาสำหรับโปรแกรมเมอร์ สายโค้ดเดอร์ และนักพัฒนาซอฟต์แวร์ทุกคน! หยั่งรู้ชะตากรรมแห่งรันไทม์ ทำนายบั๊ก ค้นพบภาษาเวทมนตร์ และถามคำถามกับลูกแก้วบั๊กวิเศษเพื่อเสริมสร้างขวัญกำลังใจในการเคลียร์บอร์ดงาน

---

## 🇹🇭 ภาษาไทย

### ✨ ฟีเจอร์หลัก (Key Features)

1. **🔮 การ์ดทำนายดวงชะตารันไทม์ (Tarot Cards)**
   - สุ่มจับไพ่เวทมนตร์ 3 มิติ 1 ใบเพื่อเช็กดวงชะตารันไทม์ในแต่ละวัน
   - แสดงระดับความปัง (Luck Level) และระดับบั๊กคุกคาม (Bug Threat)
   - แนะนำเคล็ดวิชาแก้กรรมและวิธีแก้เคล็ดเพื่อการทำงานที่ราบรื่น

2. **💻 ทำนายดวงตามภาษาเขียนโปรแกรม (Programming Language Horoscope)**
   - ค้นพบชะตากรรมประจำวันตามภาษาที่คุณรัก/ใช้บ่อย เช่น JavaScript, Python, Go, Rust, Java, C++, PHP, TypeScript, C#, และ HTML/CSS
   - บอกพลังนำโชคประจำวัน: รหัสสีนำโชค (Lucky Hex), ธีม IDE นำโชค, เครื่องดื่มเสริมพลัง และคำเตือนสิ่งกาลกิณี (เช่น อย่า Deploy วันศุกร์บ่ายสี่!)

3. **🔮 ลูกแก้วบั๊กวิเศษ (Bug Crystal Ball)**
   - ระบบตั้งจิตอธิษฐานถามลูกแก้ววิเศษเกี่ยวกับโค้ดที่รันไม่ผ่าน ปัญหาที่ทำงาน หรือชะตากรรมของบอร์ดงาน
   - แตะที่ลูกแก้วเพื่อรับคำทำนายพร้อมเอฟเฟกต์แสงวิบวับ

4. **🎵 ระบบเสียงเวทมนตร์สังเคราะห์ (Audio Synthesis Engine)**
   - เสียงเอฟเฟกต์แนว 8-bit / Magic Retro โดยใช้ **Web Audio API** แท้ๆ ไม่พึ่งพาไฟล์เสียงภายนอก

5. **🌌 ระบบอนุภาคพื้นหลังแห่งโค้ดดิ้ง (Background Canvas Particles)**
   - พื้นหลังแอนิเมชันสุดล้ำด้วยโค้ดสัญลักษณ์โปรแกรมมิ่ง (`{ }`, `</>`, `=>`, `++`, `[]`, `1`, `0`) ที่ลอยละล่องอย่างราบรื่น

## 🇺🇸 English

### ✨ Key Features

1. **🔮 Runtime Tarot Prediction**
   - Flip interactive 3D magic cards to predict your runtime luck today.
   - Shows luck levels, bug threat index, and magical remedy tips.

2. **💻 Programming Language Horoscope**
   - Tailored predictions for various languages (JS, Python, Go, Rust, Java, C++, PHP, TS, C#, HTML/CSS).
   - Reveals your Lucky Hex color, Ideal IDE Theme, Power Drink, and critical warnings.

3. **🔮 Bug Crystal Ball (Interactive Q&A)**
   - Ask any questions about your compiler errors or workspace politics, tap the crystal ball, and get the oracle's guidance.

4. **🎵 Synthesized Sound Engine**
   - Custom retro sound effects generated programmatically via the **Web Audio API**.

5. **🌌 Digital Particle Background**
   - Custom `<canvas>` particle system flowing coding symbols dynamically in the background.

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

- **Frontend Core:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Icons & Fonts:** FontAwesome 6, Google Fonts (Mitr, Fira Code, Space Grotesk)
- **Audio:** Web Audio API (Synthesized Oscillators)
- **Background Visuals:** HTML5 Canvas API

## 🚀 วิธีการใช้งานภายในเครื่อง (How to Run Locally)

1. Clone คลังเก็บโค้ดนี้:
   ```bash
   git clone git@github.com:Jakkapon-Wapakpet/codingwhice.git
   cd codingwhice
   ```

2. เปิดไฟล์ `index.html` ด้วยโปรแกรมเว็บเบราว์เซอร์ใดก็ได้ หรือใช้ Extension เช่น **Live Server** ใน VS Code เพื่อรันใน Local Server:
   ```bash
   # หากใช้งานเครื่องมือ npm ในการรันเซิร์ฟเวอร์แบบง่าย
   npx serve .
   ```
